# メールマーケティング

## ステップメールの設計

登録〜購入までの心理ステップに合わせてシナリオを設計する。

### 基本シナリオ（BtoC・商品購入）

```
Day0（登録直後）: ウェルカムメール
  → 登録のお礼 + 特典（クーポン等）+ ブランドの世界観

Day1: 問題提起メール
  → 読者が抱える悩み・課題を共感込みで提示

Day3: 解決策提示メール
  → あなたの商品・サービスがどう解決するか

Day5: 事例・証拠メール
  → お客様の声・ビフォーアフター・数値

Day7: 限定オファーメール
  → 購入者特典・期間限定割引 + CTA

Day14: フォローアップ
  → 未購入者へ：FAQ 回答・よくある不安解消
```

### ウェルカムメールは開封率が最高（50〜70%）
登録直後のメールが最も読まれる。ここで「次のステップ」を明確に伝える。

## 件名（Subject）最適化

**開封率に最も影響する要素**。30〜50 文字以内が目安。

効くパターン:
- 数字を入れる: 「売上 3 倍になった 5 つの方法」
- 問いかけ: 「なぜあなたの LP は読まれないのか？」
- 限定感: 「今だけ限定公開」「残り 3 名」
- 個人名宛て: 「{名前}さんへ」（パーソナライズ）
- 短く: 「昨日の話」（謎めかす）

避けるパターン:
- スパムワード: 「無料！」「激安」「今すぐ」（配信停止になりやすい）
- 全角感嘆符・絵文字の多用
- 件名と本文の内容が違う（開封率は上がるが信頼を失う）

## 配信頻度の目安

| 目的 | 頻度 |
|---|---|
| ステップメール（自動） | 購買シナリオに合わせる |
| 定期メルマガ | 週 1〜2 回が標準。毎日は熱量が続くときのみ |
| セール告知 | 月 1〜2 回 |
| トランザクション（購入完了等） | 即時 |

## 到達率・開封率の改善

### 到達率（Deliverability）
- 独自ドメインで送る（フリーメールは NG）
- SPF / DKIM / DMARC を設定する
- 一度に大量送信しない（ウォームアップが必要）
- バウンス（不達）メールアドレスを定期的に削除する

### 開封率向上
- 送信者名を「個人名」にする（「株式会社〇〇」より「田中太郎」の方が開かれる）
- 配信時間: 火〜木の 10 時・14 時が一般的に良い
- 定期的に非開封者をリストから削除（エンゲージメントを保つ）

## セグメント配信

全員に同じメールを送るより、セグメント別配信の方が 2〜3 倍の効果。

重要セグメント:
- **購入有無**: 未購入者・既存顧客・リピーター
- **エンゲージメント**: 直近 30 日開封・未開封
- **購入商品**: 商品カテゴリ別にアップセル
- **登録経路**: どの LP 経由か

## KPI

| 指標 | 目安 |
|---|---|
| 開封率 | 20〜30%（業界平均 20% 前後） |
| クリック率 | 2〜5% |
| コンバージョン率 | 1〜3% |
| 購読解除率 | 0.5% 以下に保つ |
