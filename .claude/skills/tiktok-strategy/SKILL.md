# TikTok 運用戦略

## アルゴリズムの仕組み

TikTok は「フォロワー数に関係なく」コンテンツが拡散される。

**評価指標（重要度順）**:
1. **視聴完了率**: 動画を最後まで見た割合。最重要指標
2. **リピート視聴**: 同じ動画を繰り返し見た割合
3. **シェア数**: 最も拡散力の高い指標
4. **コメント数**: エンゲージメントの深さ
5. **いいね数**: 基本的な反応

**配信フロー**: 小さなテストグループ（200〜500 人）に配信 → 反応が良ければ段階的に広げる

## フック（最初の 3 秒）の設計

視聴完了率を高めるには最初の 3 秒が命。

**効くフックのパターン**:
- 「〇〇している人、絶対に見てください」
- 「これをやったら売上が 3 倍になりました」
- 「普通と思っていたのに実は間違いだった話」
- 驚き・矛盾・問いかけで「続きが気になる」状態を作る

## 動画構成の基本

```
0〜3 秒: フック（スクロールを止める）
3〜10 秒: 問題提起 or 共感
10〜秒: 解決策・コンテンツ本編
ラスト 5 秒: CTA（コメントして / フォローして / リンクから）
```

長さの目安: 15〜60 秒が最もパフォーマンス良い。長尺（3〜5 分）も最近は強い。

## トレンド音源の活用

- **トレンドサウンド**: For You ページを観察して使われている音源を使う
- 音源にトレンドがつくと、その音源の動画が拡散されやすくなる
- オリジナル音楽よりトレンド音楽の方が初速が出やすい
- ただしブランドの雰囲気に合った音源を選ぶこと

## 字幕とテキスト

- 70% のユーザーが無音で視聴 → 字幕は必須
- 字幕は大きく・コントラスト強く
- テキストエフェクト（スクロール・ポップイン）で視聴を引き止める

## コメント欄の活用

- コメントに返信すると動画のエンゲージメントが上がる
- 「コメントで〇〇と書いてね」で視聴者参加型にする
- 批判コメントも返信するとアルゴリズム的に有利

## TikTok 広告との連携

- オーガニックでバズった動画を「スパーク広告」で広告配信する
- 一般人っぽいクリエイティブ（UGC 型）が広告でも効果大
- 広告感がない自然な動画の方が CTR が高い

## 投稿頻度の目安

- 最低: 週 3〜5 本
- 理想: 毎日 1〜3 本
- 量より質を意識しつつ、A/B テスト的に複数パターンを投稿する
