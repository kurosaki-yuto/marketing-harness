# Meta Ads メトリクス読解

## 主要指標の定義と目安

### CTR（クリック率）= クリック数 / インプレッション数
- 業界平均: 0.9〜1.5%
- 良い: 2.0% 以上
- 要改善: 0.5% 以下
- 認知型は 0.5% 以上で可。コンバージョン型は 1.0% 以上を目標にする

### CPC（クリック単価）= 費用 / クリック数
- 業界平均: ¥80〜¥200
- CTR が低いと CPC は上がる（逆相関）
- LP・ターゲット・業界の競合度で大きく変動

### CPM（1,000 インプレッション単価）= 費用 / インプレッション数 × 1,000
- 通常: ¥500〜¥1,500
- 上昇要因: 競合増 / ターゲット絞りすぎ / クリエイティブ疲弊

### CPA（顧客獲得単価）= 費用 / コンバージョン数
- 業種別目安は下表参照
- CPA = CPC ÷ CVR で分解して原因を特定する

### ROAS（広告費用対効果）= 売上 / 費用 × 100
- EC 最低ライン: 200%
- 黒字ライン = 100 ÷ 粗利率 ×100 で計算（粗利 50% なら 200%）

### CVR（コンバージョン率）= コンバージョン数 / クリック数
- 業界平均: 1〜3%
- 5% 超: 優秀 / 0.5% 以下: LP またはターゲット見直し

## 業種別 CPA 目安

| 業種 | CPA 目安 |
|---|---|
| EC（商品単価 ¥3,000〜¥10,000） | ¥500〜¥2,000 |
| EC（商品単価 ¥10,000〜） | ¥2,000〜¥8,000 |
| BtoB リード獲得 | ¥5,000〜¥30,000 |
| アプリインストール | ¥300〜¥1,500 |
| 店舗集客（来店 CV） | ¥1,000〜¥5,000 |
| サブスク（月額 ¥5,000〜） | ¥3,000〜¥15,000 |

## 学習フェーズの判断

- **条件**: 直近 7 日で 50 コンバージョン未満のとき「学習中」になる
- **対応**: 学習完了まで予算・ターゲット・クリエイティブを変更しない
- 50 CV に達しない場合はコンバージョンイベントを上位ファネルに変更（購入 → カート追加 → 閲覧）

## 指標悪化時の切り分け

| 症状 | 疑うべき原因 | 確認箇所 |
|---|---|---|
| CTR 低下 | クリエイティブ疲弊・ターゲット不一致 | フリークエンシー・クリエイティブ別 CTR |
| CPC 上昇 | 競合増・CTR 低下 | オークション洞察・CTR |
| CVR 低下 | LP 品質低下・ターゲット変化 | LP ヒートマップ・ターゲット詳細 |
| CPA 上昇 | CPC 上昇 or CVR 低下 | CPC × CVR の積で分解 |
| ROAS 低下 | CPA 上昇 or 客単価低下 | 注文あたり売上・CPA |
