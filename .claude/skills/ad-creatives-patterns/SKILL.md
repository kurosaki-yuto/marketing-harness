# 広告クリエイティブ 勝ちパターン

## フォーマット別の用途と強み

| フォーマット | 得意なこと | 弱点 |
|---|---|---|
| 静止画（1:1 / 4:5） | 素材が少ない、即製、テスト向き | 止める力は動画より低い |
| 動画（15〜30 秒） | 世界観・使い方の説明 | 制作コスト・時間がかかる |
| リール / ショート（9:16） | 若年層・バイラル・低 CPM | スキップ率が高い |
| カルーセル | 複数商品・ステップ説明 | スワイプしてもらえないと効果薄 |
| コレクション（EC） | モバイル購買・商品発見 | CMS / カタログ整備が必要 |

## ファーストフレーム（最初の 3 秒）の法則

**止める要素（優先度順）:**
1. 人物の顔 / 表情（共感・感情移入）
2. 動き・変化（視線が引き寄せられる）
3. 強い色のコントラスト
4. テキストオーバーレイ（大きなフォントで問いかけ）

**NG パターン:**
- ロゴからスタート（見飽きている）
- 暗い / 動きのない映像（スクロールで埋もれる）
- 3 秒以内に何の広告かわからない

## クリエイティブパターン分類

### A. UGC 型（ユーザー投稿風）
- スマホ縦撮り・手持ちカメラ感
- 「友達に送るメッセージ」的な話し言葉テキスト
- 広告感を消すことでスキップ率が下がる
- 効果: CTR +20〜40% の前例多数

### B. ビフォーアフター型
- 最初の 3 秒で「前の状態（問題）」を見せる
- 5〜10 秒で「後の状態（変化）」を見せる
- 変化が大きいほどクリック意欲が上がる

### C. デモ / ハウツー型
- 「3ステップで〇〇できる方法」
- 商品の使い方・操作を実演
- スクロールを止める力は弱いが CVR が高い

### D. テキスト主体型（静止画）
- 背景シンプル + 大きなテキスト 1 行
- 問いかけや強い言葉を正面に出す
- 例:「まだ月2万円も払ってますか？」

### E. 社会的証明型
- 星評価・レビュー数をビジュアルに含める
- インフルエンサーのセリフ風引用
- 「〇〇件の口コミ」数字の提示

## クリエイティブの疲弊サイン

| 指標 | 疲弊ライン | 対応 |
|---|---|---|
| フリークエンシー | 3.5 超 | 差し替え or ターゲット拡張 |
| CTR | 直近比 -30% 以上 | 差し替え必須 |
| フック率（3 秒視聴率） | 40% 未満 | ファーストフレームの変更 |
| CPM | 直近比 +30% 以上 | クリエイティブ or 競合季節要因 |

## テスト設計の原則

- **1テスト = 1変数**（見出しを変えるならビジュアルは固定）
- **テスト期間**: 統計的有意に達するまで最低 7〜14 日
- **サンプルサイズ**: 広告セットあたり 2〜4 本（それ以上は学習が分散）
- **勝ち判定**: CTR, CVR, CPA のうち主目標指標で +15% 以上かつ信頼区間 90%

## サムネイル / カバー画像のポイント

- 16:9 動画でも 1:1 サムネイルを別途用意（フィード表示で有利）
- 人物が画面中央に来るよう構図を設計
- テキスト占有率 20% 以下（Meta ガイドライン）

## クリエイティブ素材の収集・活用

1. 顧客のレビューや SNS 投稿（UGC）を許可取りして活用
2. Canva / CapCut で UGC 風に再編集
3. 商品撮影は「使用シーン」重視（パッケージ単体 < 使っている人）
4. A/Bテストで勝ったクリエイティブの特徴をナレッジに追加
